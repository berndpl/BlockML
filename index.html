BlockML is a open source markup language for scientific documents. It combines the most useful functions of LaTeX with the simplicity and advantages of lightweight markup languages like Markdown. Perfect for documentation, complex publication and scientific work being shared on the web.

  • Intuitive block based syntax
  • Advanced scientific features (mathematica, bibliographic references), but easy to use
  • Perfect for being published on the web, great to read on tablets
  • Mac OS X client for streamlined documentation generation

Getting Started

1

Create a text file written with the easy to learn Block ML Syntax.

document.txt

fp[

title[Example Document Title]

h4[Author Name]

]


toc[Table of Contents]


sec[First Section][

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

sec[First Subsection][

math[$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}$$]

]/* First Subsection */

]/* First Section */

2

Drag and Drop your Block ML text file onto the BlockML OSX App to generate your final documentation.

document.html

Example Document Title

Author Name

Table of Contents

  • 1 First Section[⚐]
    • 1.1 First Subsection[⚐]

1 First Section

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua.

1.1 First Subsection

$$x = {-b \pm \sqrt{b^2-4ac} \over 2a}$$

Done! You just created your first perfectly styled documentation, ready to be shared.

Documentation

The syntax of BlockML is slightly S-Expression inspired and designed to be consistent and memorable. A BlockML markup element consists of a selector, who defines the type of the element and one or more blocks, which embed the text you want to mark up: selector[block]

DOCUMENTATION

BlockML provides a couple of features wich are especially needed for academic documents, like sections, tables of contents, bibliographic references or captions. BlockML also provides features wich are common for lightweight markup languages, like links, headings, images and so on.
